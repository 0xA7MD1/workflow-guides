- عند الضغط على "ابدأ"، يتم التوجيه إلى صفحة `/workflow/[typeId]` مع تمرير اسم المشروع عبر query parameter أو context.

- تجلب بيانات النوع من `workflows.json`.
- تعرض المراحل والخطوات بنفس التصميم السابق (مع إمكانية فتح/إغلاق).

- تحفظ حالة فتح المراحل والخطوات في localStorage.
- [x] إنشاء دوال في `lib/storage.ts` لإدارة قائمة المشاريع (إضافة، جلب، تحديث، حذف) مع حفظ كل مشروع بحالته الخاصة (الاسم، النوع، حالة المراحل والخطوات).
- [x] تعديل الصفحة الرئيسية (`app/page.tsx`) لتعرض قائمة المشاريع المخزنة (بطاقات تحتوي على اسم المشروع ونوعه وأيقونته).
- [x] إضافة زر "مشروع جديد" يفتح Modal (نفس الموجود سابقاً) لإنشاء مشروع جديد مع اختيار النوع وإدخال الاسم، ثم حفظه وعرضه في القائمة.

- [x] عند إجراء أي تغيير في صفحة المراحل (فتح مرحلة، إكمال خطوة)، تحديث `localStorage` للمشروع الحالي.
